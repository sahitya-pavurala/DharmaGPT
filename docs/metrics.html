Every step uses a specific model for a specific purpose. Data pipeline models and query pipeline models are completely separate — transcription and translation happen once offline; retrieval and generation happen at query time.

Architecture

End-to-End Flow

From a raw audio recording to a scored, citation-grounded answer.

Data Pipeline (offline, once)
1
Audio → 29s chunks
ffmpeg splits long pravachanas. 29s gives deliberate buffer under Sarvam's duration limit.
2
Transcribe → Telugu text
Sarvam saaras:v3 STT. Each chunk becomes one JSONL record.
3
Translate → English
Anthropic → Ollama → IndicTrans2 fallback chain. Every failure is logged with a reason field.
4
Embed → upsert to pgvector
OpenAI text-embedding-3-large at 3072 dimensions. Cosine similarity retrieval.
Query Pipeline (every request)
1
Embed query
Same OpenAI embedding model ensures vector space alignment.
2
Retrieve top-5 chunks
pgvector cosine similarity. Min score threshold: 0.35. Max context: 6000 chars.
3
Generate answer
Claude claude-sonnet-4 with mode-specific prompts. Citations required in output.
4
Score response
Two Sarvam judge calls. Primary: relevance + context. Secondary: faithfulness + citations.

Model Responses Across Query Modes

The same query engine, different response structures. Each mode has explicit compliance rules checked by regex — no LLM required for structural validation.

GUIDANCE MODE
PASS · 0.83 claude-sonnet-4
Query
How should I deal with anger and frustration in daily life?
Answer
The Ramayana offers a profound teaching on this through Rama's conduct during his exile. Despite facing unjust exile, Rama maintained equanimity — he did not rage against Kaikeyi or Dasharatha, but instead reflected on his dharmic duty. [Valmiki Ramayana, Ayodhya Kanda, Sarga 19]

The text teaches that anger arises when we believe a situation should be different from what it is. Rama's path was to fully accept the situation while choosing his response consciously — fulfilling his role as a devoted son while remaining inwardly steady.

What would it feel like to separate the situation from your reaction to it — even for one difficult moment today?
Faithfulness: 0.85 Relevance: 0.90 Context use: 0.78 Citations: 0.88
Source: Valmiki Ramayana, Ayodhya Kanda · Retrieved score: 0.847
SCHOLAR MODE
PASS · 0.79 claude-sonnet-4
Query
What is the significance of Hanuman crossing the ocean in the Sundara Kanda?
Answer
Hanuman's ocean crossing in Sundara Kanda (Sargas 1–17) is structurally and thematically the pivot of the Ramayana. The feat operates on three registers simultaneously:

Narrative function: It bridges the search phase (Kishkindha Kanda) and the rescue phase (Yuddha Kanda), establishing that Sita is alive and identifiable by Rama's ring. [Valmiki Ramayana, Sundara Kanda, Sarga 15]

Theological register: Hanuman's leap enacts the principle of bhakti as the mechanism that removes the impossible. His invocation of Rama's name before the leap is textually explicit — the name enables the crossing, not physical capability alone.

Structural parallel: The ocean mirrors Lanka's isolation. Crossing it collapses Ravana's presumed safety, foreshadowing the siege of Yuddha Kanda.
Faithfulness: 0.80 Relevance: 0.88 Context use: 0.72 Citations: 0.75
Sources: Valmiki Ramayana, Sundara Kanda, Sargas 1–17 · Retrieved score: 0.861
